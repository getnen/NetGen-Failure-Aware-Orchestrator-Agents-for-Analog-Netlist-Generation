**Topology Summary**
Use one NMOS as a low-side constant current sink with a resistive pull-up load. `Vout` is the shared drain/load node. The resistor connects from `VDD` to `Vout`; the NMOS connects from `Vout` to ground. The gate is driven directly by `Vbias`.

**Device/Function List**

- `Vbias`: DC bias source named explicitly as `Vbias`, connected `Vbias 0`, value `DC 1.0`.
- `R`: load resistor from `VDD` to `Vout`, nominal `10k`.

**Bias/Reference Strategy**
`VDD=5 V` establishes the pull-up rail. `Vbias=1.0 V` gives the NMOS `Vgs=1.0 V`, above threshold, so the transistor sinks current from `Vout` through its channel to ground. The resistor converts that sink current into a DC output voltage at `Vout`.

**Signal or Feedback Path**
There is no active feedback or mirror reference branch. The DC path is:
`VDD -> R -> Vout -> NMOS drain/source -> 0`.

`Vbias` only controls the NMOS gate and should not be loaded except by the MOS gate.
