**Topology Summary**

Use a standard static CMOS inverter powered from a 5 V rail. `Vout` is the shared drain/output node between the pull-up PMOS and pull-down NMOS. `Vin` drives both transistor gates directly.

When `Vin = 0 V`, PMOS is on and NMOS is off, so `Vout` is pulled to `VDD`. When `Vin = 5 V`, NMOS is on and PMOS is off, so `Vout` is pulled to ground.

**Device/Function List**

- `VDD` supply: independent DC voltage source from node `VDD` to `0`, value `5 V`.
- Input source `Vin`: independent voltage source from node `Vin` to `0`, used as the `.dc` sweep source.
- NMOS pull-down:
  - Drain: `Vout`
  - Gate: `Vin`
  - Source: `0`
  - Bulk: `0`
  - Model: `NMOS`, threshold about `0.7 V`
- PMOS pull-up:
  - Source: `VDD`
  - Gate: `Vin`
  - Drain: `Vout`
  - Bulk: `VDD`
  - Model: `PMOS`, threshold about `-0.7 V`

**Bias/Reference Strategy**

`VDD` is fixed at `5 V` relative to ground node `0`. `Vin` is explicitly driven by its own independent voltage source and swept from `0 V` to `5 V`. Bulk terminals should be tied to their respective sources to avoid body-effect surprises in this simple inverter.

**Signal Or Feedback Path**

There is no feedback path. This is a direct combinational inverter: `Vin` controls both gates, and `Vout` is formed by the complementary pull-up/pull-down conduction path between `VDD` and `0`.

**SPICE/Netlist Constraints**

- Final file must be named `p7.cir`.
- Preserve node names exactly: `Vin`, `Vout`.
- Use exactly one NMOS and one PMOS transistor.
- Include an independent DC input source named `Vin` so `.dc Vin 0 5 0.1` sweeps the source correctly.
- Include primitive MOS models with thresholds:
  - `NMOS` with `VTO = 0.7`
  - `PMOS` with `VTO = -0.7`
- Use primitive MOSFET devices only; no behavioral inverter or controlled-source shortcut.

**Failure Risks To Avoid**

- Do not leave `Vin` floating; it must be driven by an independent source named `Vin`.
- Do not accidentally make `Vin` only a node name while the voltage source has another name, because `.dc Vin ...` must reference the source name.
- Do not swap PMOS source/bulk away from `VDD`; PMOS source and bulk should both connect to `VDD`.
- Do not swap NMOS source/bulk away from ground; NMOS source and bulk should both connect to `0`.
- Do not use extra transistors, resistive loads, behavioral sources, or logic primitives.
- Ensure `Vout` is only the shared output node between the PMOS drain and NMOS drain, not tied directly to `VDD` or ground.
