=== TOPOLOGY PLAN ===
**Topology Summary**

Use a standard static CMOS inverter powered from a 5 V rail. `Vout` is the shared drain/output node between the pull-up PMOS and pull-down NMOS. `Vin` drives both transistor gates directly.

When `Vin = 0 V`, PMOS is on and NMOS is off, so `Vout` is pulled to `VDD`. When `Vin = 5 V`, NMOS is on and PMOS is off, so `Vout` is pulled to ground.

**Device/Function List**

- `VDD` supply: independent DC voltage source from node `VDD` to `0`, value `5 V`.
- Input source `Vin`: independent voltage source from node `Vin` to `0`, used as the `.dc` sweep source.
- NMOS pull-down:
  - Drain: `Vout`
  - Gate: `Vin`
  - Source: `0`
  - Bulk: `0`
  - Model: `NMOS`, threshold about `0.7 V`
- PMOS pull-up:
  - Source: `VDD`
  - Gate: `Vin`
  - Drain: `Vout`
  - Bulk: `VDD`
  - Model: `PMOS`, threshold about `-0.7 V`

**Bias/Reference Strategy**

`VDD` is fixed at `5 V` relative to ground node `0`. `Vin` is explicitly driven by its own independent voltage source and swept from `0 V` to `5 V`. Bulk terminals should be tied to their respective sources to avoid body-effect surprises in this simple inverter.

**Signal Or Feedback Path**

There is no feedback path. This is a direct combinational inverter: `Vin` controls both gates, and `Vout` is formed by the complementary pull-up/pull-down conduction path between `VDD` and `0`.
