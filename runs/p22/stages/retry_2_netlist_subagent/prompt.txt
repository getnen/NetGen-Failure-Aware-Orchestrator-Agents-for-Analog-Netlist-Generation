=== TOPOLOGY PLAN ===
**Topology Summary**

Use a 5 V single-supply behavioral op-amp RC phase-shift oscillator. The output node must be exactly `Vout`. The oscillator core is a high-gain inverting amplifier centered at `Vcm = 2.5 V`, with a three-section RC phase-shift ladder feeding `Vout` back to the inverting input `Vinn`. The non-inverting input `Vinp` is fixed at `Vcm`.

The design should have startup loop gain comfortably above unity. Since the three-section RC ladder attenuates by about 1/29 at oscillation, the inverting gain stage should target about 35 to 50 small-signal gain, then rely on supply-limited behavioral output or soft limiting to stabilize amplitude.

**Device/Function List**

- `VDD` supply: independent 5 V DC source from `VDD` to `0`.
- Mid-supply divider:
  - Equal resistors from `VDD` to `Vcm` and `Vcm` to `0`.
  - Bypass capacitor from `Vcm` to `0`.
- Behavioral op-amp-like gain block:
  - Explicit input nodes `Vinp` and `Vinn`.
  - `Vinp` tied to `Vcm`.
  - Output exactly `Vout`.
  - Output limited between 0 V and 5 V, preferably with smooth limiting around rails.
- Three RC phase-shift sections:
  - Use equal values, for example `R = 10k`, `C = 10nF`.
  - Intermediate nodes such as `n1`, `n2`, `n3`.
  - Final ladder node should drive `Vinn`.
- Bias resistor:
  - Large resistor from `Vinn` or `n3` to `Vcm`, for example 1 MOhm to 10 MOhm, to define DC without strongly loading the ladder.
- Gain-setting network:
  - Inverting gain equivalent around 35 to 50.
  - Avoid using values that significantly load the RC ladder.

**Bias/Reference Strategy**

All oscillator signal nodes should be DC-referenced to `Vcm`, not ground. The RC ladder resistors should return to `Vcm` so the AC phase shift occurs around the 2.5 V common-mode level. `Vinp` should be directly biased to `Vcm`.

Use startup asymmetry with `.ic`, for example slight mismatch between `Vinp` and `Vinn`, or initial ladder-node voltages such as `n1`, `n2`, `n3` slightly offset from 2.5 V. Do not rely only on numerical noise.

**Signal Or Feedback Path**

The intended loop is:

`Vout -> RC ladder section 1 -> n1 -> RC ladder section 2 -> n2 -> RC ladder section 3 -> n3/Vinn -> high-gain inverting amplifier -> Vout`

The RC ladder supplies about 180 degrees phase shift near the oscillation frequency. The inverting amplifier supplies the additional 180 degrees. For `R = 10k` and `C = 10nF`, expect a visible low-kHz or sub-kHz oscillation, roughly 600 Hz to 1 kHz depending on loading.

**SPICE/Netlist Constraints**

- Final output node name must be exactly `Vout`.
- Include exactly `.tran 1u 5m`.
- No external input source is needed.
- Do not leave `Vinn`, `Vinp`, `n1`, `n2`, `n3`, `Vcm`, or `Vout` floating.

- Use `.ic` or startup-compatible initial capacitor conditions.

**Failure Risks To Avoid**

- Gain too low: previous failure showed only about 20 mV output amplitude, so use gain above 29, preferably 35 to 50.
- Ladder referenced to ground instead of `Vcm`, which breaks single-supply operation.
- Excessively loading the RC ladder with small gain-setting or bias resistors.
- Hard saturation at 0 V or 5 V without recovery; use smooth limiting or enough feedback biasing so `Vout` swings around `Vcm`.
- Floating final ladder node or undefined `Vinn` DC point.
- Over-damped startup from no initial mismatch or too much shunt loading.
- Incorrect output node name; the checker requires `Vout`.
