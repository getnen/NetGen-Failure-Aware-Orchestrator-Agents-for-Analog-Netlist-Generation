Objective: Create a 5 V single-supply RC phase-shift oscillator whose output node `Vout` develops a sustained visible periodic waveform during `.tran 1u 5m`. There are no external input nodes. The final netlist must include `Vout`.

Topology: Use an inverting high-gain amplifier stage built from MOSFETs or an op-amp-like transistor gain block, biased around mid-supply, with a three-section RC phase-shift feedback network from `Vout` back to the amplifier input. The RC ladder should provide approximately 180 degrees phase shift near the oscillation frequency, while the inverting amplifier supplies the other 180 degrees. Ensure loop gain is comfortably greater than unity at startup.

Required devices:
- 5 V supply source: `VDD` to ground = 5 V.
- Mid-supply bias/reference source or divider creating `VREF` near 2.5 V.
- Inverting gain stage with output node exactly named `Vout`.
- Three cascaded RC sections in the feedback path from `Vout` to the amplifier input node.
- Bias resistors tying the amplifier input operating point to `VREF` so the single-supply circuit starts near mid-rail.
- MOSFETs may use provided models `NMOS` and `PMOS`; connect each MOS bulk to its source.
- Add startup assistance using `.ic` on internal feedback/amplifier input nodes or a slight initial mismatch between internal differential input nodes if using an op-amp-style gain stage.

Key interconnectivity:
- `Vout` feeds RC section 1, then RC section 2, then RC section 3.
- The final RC ladder node drives the inverting/high-gain input of the amplifier.
- The amplifier output is `Vout`.
- The gain stage must be biased so `Vout` rests near mid-supply at operating point but can swing above and below that midpoint.
- The RC network should be AC-coupled or DC-biased so no feedback node floats; every internal RC node needs a DC path to `VREF` or ground.
- Choose R and C values that place oscillation within the 5 ms transient window, for example tens of kilohertz or lower. A practical target is roughly 5 kHz to 20 kHz so several cycles appear in `.tran 1u 5m`.

Biasing assumptions:
- Use `VDD = 5 V`.
- Use `VREF = 2.5 V` as the amplifier common-mode/virtual-ground reference.
- MOS thresholds are `NMOS Vth=0.7 V` and `PMOS Vth=-0.7 V`; size transistors nominally but make the gain stage strong enough that startup loop gain exceeds the RC ladder attenuation.
- If a differential op-amp-like stage is used, bias its positive input at `VREF` and drive its negative input from the RC feedback ladder, or vice versa as needed to preserve net inverting loop behavior.
- Include startup initial conditions such as a small offset on the final RC node or amplifier input node so oscillation grows from transient startup rather than remaining perfectly balanced.

Required analysis:
- Include exactly the required transient analysis: `.tran 1u 5m`.

Expected behavior/output constraints:
- `Vout` must oscillate visibly in transient simulation.
- The latter portion of the 5 ms waveform should show sustained periodic behavior rather than a decaying transient.
- `Vout` should remain within the 0 V to 5 V supply rails and preferably swing around the 2.5 V midpoint.
- The RC ladder internal nodes should show progressively phase-shifted sinelike waveforms relative to `Vout`.
- The final netlist must not leave hidden floating input nodes, must not depend on external stimulus inputs, and must include node `Vout` exactly.
