**Topology summary**
Use a two-stage CMOS voltage amplifier on `VDD = 5 V`. Stage 1 is an NMOS common-source gain device with PMOS current-source/active load, producing intermediate node `N1`. Stage 2 is another NMOS common-source gain device with PMOS active load, producing output node `Vout`. Add Miller compensation capacitor `Cc` from `Vout` back to `N1`. Include explicit load resistor `RL` in the 10k-100k range, preferably `50k`, from `Vout` to `0`.

**Device/function list**
- `VDD`: independent DC source from `VDD` to `0`, value `5 V`.
- `Vin`: independent voltage source directly driving node `Vin`, recommended `DC 1.5 AC 1`.
- `M1`: NMOS input/common-source transistor. Drain `N1`, gate `Vin`, source/bulk `0`.
- `M2`: PMOS active load for stage 1. Source/bulk `VDD`, drain `N1`, gate `Vbp`.
- `M3`: NMOS second-stage common-source transistor. Drain `Vout`, gate `N1`, source/bulk `0`.
- `M4`: PMOS active load for stage 2. Source/bulk `VDD`, drain `Vout`, gate `Vbp`.
- `VBP`: independent DC bias source from `Vbp` to `0`, nominal around `3.8 V`.
- `Cc`: Miller compensation capacitor from `Vout` to `N1`, nominal `2pF`.
- `RL`: resistive load from `Vout` to `0`, nominal `50k`.

**Bias/reference strategy**
Use explicit, non-floating references: `VDD` fixes the supply, `Vin` fixes the input DC operating point, and `VBP` fixes the PMOS gate bias. With PMOS sources at `VDD`, `Vbp ≈ 3.8 V` gives `VSG ≈ 1.2 V`, turning PMOS loads on with moderate current. `Vin = 1.5 V` gives the input NMOS enough overdrive above `Vth = 0.7 V`.

The netlist writer should check `.op` results and adjust `Vbp`, transistor sizing only if necessary, or resistor value within spec so `N1` is high enough to turn on `M3` and both `N1` and `Vout` remain away from rails. Target saturation for `M1` and `M3`; PMOS loads should also remain in a current-source-like region.

**Signal or feedback path**
Small-signal input enters at `Vin`, modulates `M1`, and creates an inverted voltage gain at `N1`. `N1` drives the gate of `M3`, producing a second inversion at `Vout`, so total low-frequency voltage gain is non-inverting from `Vin` to `Vout`.

`Cc` from `Vout` to `N1` is the Miller compensation path. It provides dominant-pole compensation around the high-gain second stage and should shape the AC response for `.ac dec 20 1 1e9`.

**SPICE/netlist constraints**
- Save final file as `p14.cir`.
- Preserve exact node names `Vin` and `Vout`.
- `Vin` must be driven by an independent source with both DC and AC values.
- `Vout` must connect to `M3` drain, `M4` drain, `Cc`, and `RL`.
- Include `.op`.
- Include `.ac dec 20 1 1e9`.
- Use the provided MOSFET models directly; no subcircuits or placeholder models.
- Connect every MOS bulk to its source.
- Keep explicit nodes `Vin`, `Vout`, `N1`, `Vbp`, `VDD`, and `0`.

**Failure risks to avoid**
- Do not leave `Vbp` floating or generated by an undefined bias node.
- Do not omit `AC 1` on the `Vin` source, or AC analysis will show no meaningful gain.
- Do not connect PMOS bulks to ground; PMOS bulk must be tied to `VDD`.
- Do not place `Cc` from `Vout` to ground; it must connect from `Vout` back to `N1`.
- Avoid biasing `Vbp` too low, which can overdrive PMOS loads and pin nodes high.
- Avoid biasing `Vbp` too high, which can starve the amplifier and pin nodes low.
- Avoid `Vout` operating near `0` or `5 V`, because output swing and small-signal gain will be poor.
- Ensure `RL` is within `10k` to `100k`; recommended `50k`.
