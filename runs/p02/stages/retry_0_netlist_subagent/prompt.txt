**Topology Summary**
Three cascaded NMOS common-source voltage-gain stages, all powered from `VDD = 5 V`. Each stage uses a grounded-source NMOS with a drain resistor to `VDD`. The first gate is driven by the external input node `Vin`; the third drain is the external output node `Vout`.

Internal stage outputs:
- Stage 1 drain: `n1`
- Stage 2 drain: `n2`
- Stage 3 drain: `Vout`

- Input source: independent voltage source driving exact node `Vin` relative to `0`, with DC bias around `1.5 V` and `AC 1`.
- `M1`: NMOS common-source input stage, drain `n1`, gate `Vin`, source `0`, bulk `0`, model `NMOS`, `W=50u`, `L=1u`.
- `M2`: NMOS second gain stage, drain `n2`, gate `n1`, source `0`, bulk `0`, model `NMOS`, `W=50u`, `L=1u`.
- `M3`: NMOS third gain/output stage, drain `Vout`, gate `n2`, source `0`, bulk `0`, model `NMOS`, `W=50u`, `L=1u`.
- `R1`: resistive drain load from `VDD` to `n1`, nominally `47k`.
- `R2`: resistive drain load from `VDD` to `n2`, nominally `47k`.
- `R3`: resistive drain load from `VDD` to `Vout`, nominally `47k`.

**Bias/Reference Strategy**
Use ground node `0` as the only reference. Tie all NMOS sources and bulks directly to `0`. Bias `Vin` with an explicit voltage source at approximately `1.5 V` DC so `M1` is on and `.ac` has a valid operating point.

Because stages are DC-coupled, `n1` biases the gate of `M2`, and `n2` biases the gate of `M3`. The netlist writer should expect `.op` to determine whether all three stages conduct. Drain voltages should ideally remain away from both `0 V` and `5 V`, but this simple DC-coupled three-stage chain may bias unevenly depending on the `NMOS` model.

**Signal or Feedback Path**
Signal path is strictly feed-forward:

`Vin -> M1 gate -> n1 -> M2 gate -> n2 -> M3 gate -> Vout`

Each common-source stage inverts, so three stages produce an overall inverting gain from `Vin` to `Vout`. There is no intentional global or local feedback path other than intrinsic MOS capacitances and device parasitics from the model.
