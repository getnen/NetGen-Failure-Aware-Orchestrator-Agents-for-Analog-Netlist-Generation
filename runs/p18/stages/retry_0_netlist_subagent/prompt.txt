=== TOPOLOGY PLAN ===
**Topology Summary**

Single-stage NMOS differential common-source amplifier with PMOS current-mirror active load and NMOS tail current source. The requested ports must remain exactly `Vinp`, `Vinn`, and `Vout`. The output `Vout` is single-ended at the drain of the right input NMOS and right PMOS mirror load.

**Device/Function List**

- `VDD`: 5 V DC supply from `VDD` to `0`.
- `Vinp`: 2.5 V DC input source from `Vinp` to `0`.
- `Vinn`: 2.5 V DC input source from `Vinn` to `0`.
- `Vbias`: 1.2 V DC tail bias source from `Vbias_tail` to `0`.

- `M1`: NMOS left input transistor. Drain `Nmirror`, gate `Vinp`, source/bulk `Ntail`.
- `M2`: NMOS right input transistor. Drain `Vout`, gate `Vinn`, source/bulk `Ntail`.
- `M3`: PMOS diode-connected active load. Source/bulk `VDD`, gate/drain `Nmirror`.
- `M4`: PMOS mirror load. Source/bulk `VDD`, gate `Nmirror`, drain `Vout`.
- `M5`: NMOS tail current source. Drain `Ntail`, gate `Vbias_tail`, source/bulk `0`.

**Bias/Reference Strategy**

At `Vinp=Vinn=2.5V`, `M1` and `M2` should share the tail current approximately equally. `M5` is gate-biased by `Vbias_tail=1.2V`, producing a nonzero tail current while allowing `Ntail` to rise above ground.

`M3` is diode-connected, so the left branch current establishes the PMOS mirror gate voltage at `Nmirror`. `M4` mirrors this current into the right/output branch. `Vout` should bias somewhere between `0` and `VDD`, not pinned to either rail.

**Signal or Feedback Path**

Differential input action occurs through the shared source node `Ntail`. A change in `Vinp` changes current through `M1`, which changes `Nmirror` and therefore the PMOS mirror gate drive for `M4`. This creates active-load current steering into `Vout`. `Vinn` directly controls `M2`, which sinks current from `Vout`.

There is local diode feedback around `M3` because its gate and drain are tied at `Nmirror`. This stabilizes the PMOS mirror bias but is not global amplifier feedback.
