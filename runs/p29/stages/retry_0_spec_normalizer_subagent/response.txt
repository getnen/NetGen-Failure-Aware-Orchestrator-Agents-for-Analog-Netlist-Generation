Objective:
Implement one nominal, runnable Wien-bridge voltage-controlled oscillator that produces a sustained near-sinusoidal output at node Vout. The oscillation frequency must be set by a Wien-bridge RC network whose effective resistance is explicitly modulated by Vctrl. The named input nodes Vin and Vctrl must both appear in the final netlist and must each be driven by explicit independent sources. Because this is an op-amp-like oscillator, keep Vinp, Vinn, and Vout as explicit nodes in the implementation.

Topology:
Use a single-supply, primitive-MOS two-stage op-amp core with explicit non-inverting input node Vinp, inverting input node Vinn, and output node Vout. Bias the whole oscillator from VDD = 12 V and VSS = 0 V, with a generated mid-supply reference VREF = 6 V used as the small-signal ground for the Wien bridge and for input biasing.

Connect the Wien bridge from Vout back to Vinp:
- Series arm from Vout to Vinp: capacitor C1 in series with a Vctrl-sensitive resistor element R1(Vctrl).
- Shunt arm from Vinp to VREF: resistor R2(Vctrl) in parallel with capacitor C2.
Choose C1 = C2 and make R1(Vctrl) and R2(Vctrl) matched so the classic Wien condition is preserved while frequency varies mainly as 1 / (2*pi*R(Vctrl)*C).

Use one explicit control mechanism only:
- Realize R1(Vctrl) and R2(Vctrl) as matched MOS triode-region variable resistors, each optionally padded with a small fixed series resistor so the resistance stays bounded and startup remains numerically stable.
- Drive the gates of both variable-resistor devices from Vctrl through a simple bias translation network if needed so the devices stay in triode over the expected signal swing around VREF.
- Do not use diode stabilizers, behavioral resistors, switches, or source sweeps.

Set the amplifier closed-loop gain slightly above 3 for startup:
- Feed Vinn from a resistive negative-feedback divider referenced to VREF.
- Choose the divider ratio for nominal non-inverting gain around 3.1 to 3.3 at the chosen Vctrl operating point so oscillation starts reliably.
- Let amplitude settle by the primitive amplifier’s finite output swing / gm compression rather than diode AGC, since diodes are forbidden.

Vin handling:
- Vin is still required as an explicit input node even though this is an autonomous oscillator.
- Do not leave Vin floating. Drive Vin with an explicit independent source.
- Use Vin as a weak startup injection into Vinp through a large resistor or a small coupling capacitor, so the named node is electrically present without dominating steady-state oscillation.
- The nominal design should still oscillate with Vin set to 0 V DC plus a very small transient perturbation or tiny sine component.

Required devices:
- MOSFETs only for the op-amp core and the Vctrl-sensitive resistor pair.
- Op-amp core: NMOS differential input pair, PMOS active-mirror load, NMOS or PMOS second gain stage, bias current sources/mirrors, and a Miller compensation capacitor if needed for stable amplifier behavior.
- Passive devices: matched capacitors C1 and C2, feedback resistors around the amplifier, VREF divider/buffer resistors, startup injection resistor from Vin, and any small source-degeneration or bounding resistors needed for the triode resistors.
- Independent sources: VDD supply source, explicit source for Vctrl, and explicit source for Vin.

Biasing assumptions:
- Single-supply operation: VDD = 12 V, VSS = 0 V.
- Mid-supply reference: VREF = 6 V generated by a resistor divider and lightly decoupled; use it to bias Vinp-side RC elements and the amplifier feedback network.
- MOS bulk connections: tie bulk to source for discrete idealized primitive use unless a shared-substrate process assumption is explicitly chosen in the final deck.
- Size transistors modestly, not aggressively. Use a reasonably strong input pair and output stage so the loop has enough gain at the intended oscillation frequency, but avoid huge devices that overload the RC bridge.
- Choose the nominal Vctrl so the variable resistors sit in a practical middle range, giving a nominal oscillation frequency in a comfortable transient-simulation band such as low-kHz to tens-of-kHz.

Key interconnectivity:
- Vout drives the Wien bridge.
- Wien bridge output returns to Vinp.
- Vin weakly injects into Vinp through a large resistor or small capacitor.
- Vinn receives the resistive feedback divider from Vout to VREF that sets non-inverting gain slightly above 3.
- Vctrl biases the matched MOS variable-resistor pair in the Wien network only; it is the sole intended frequency-control path.
- Vout is the required output node and must be directly observable in transient analysis.

Expected behavior:
- At the nominal Vctrl setting, transient analysis should show startup from noise/perturbation and then sustained oscillation at Vout.
- The waveform at Vout should be approximately sinusoidal and centered around VREF in the internal signal path, with output swing limited by the op-amp core’s headroom.
- Changing Vctrl in separate external runs should change the effective bridge resistance and therefore shift oscillation frequency monotonically. This frequency-versus-control behavior must come from the matched Vctrl-sensitive RC elements, not from any internal sweep construct.

Analyses required:
- Operating point to confirm all bias nodes, especially VREF, Vinp, Vinn, op-amp tail currents, and the Vctrl-controlled resistor devices, are in sensible regions.
- Transient analysis long enough to show startup and several steady-state cycles at one nominal Vctrl. Use a small startup perturbation if needed, but no multi-run control blocks.
- Optional Fourier or zero-crossing measurement is acceptable in the final deck to estimate nominal oscillation frequency, but the deck must remain a single nominal run.

Output constraints:
- Final netlist filename must be p29.cir.
- The final netlist must contain the exact nodes Vin, Vctrl, and Vout.
- Because the topology is op-amp-like, Vinp and Vinn must also be explicit net names.
- No .step, .alter, or embedded multi-run control logic.
- Use primitive-device SPICE only; no ideal op-amp macro, no behavioral VCVS-based oscillator core, and no diode limiter.
