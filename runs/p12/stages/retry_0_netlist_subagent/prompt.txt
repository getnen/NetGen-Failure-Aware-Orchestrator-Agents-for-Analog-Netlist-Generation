  Purpose: low-pass series element; isolates `Vout` from `N1`.

`Vin` -> `C1` -> `N1` -> `R2` -> `Vout`

Shunt return paths:
- `N1` -> `R1` -> `0`
- `Vout` -> `C2` -> `0`
