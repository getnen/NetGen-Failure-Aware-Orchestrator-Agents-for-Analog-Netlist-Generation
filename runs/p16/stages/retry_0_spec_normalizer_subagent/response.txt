Objective: Create a 5 V single-supply differential opamp core with an NMOS differential input pair, PMOS active current-mirror load, NMOS tail current source, and two output nodes named exactly Voutp and Vout. The circuit must support a valid DC operating point where every MOSFET conducts.

Topology: Use NMOS M1/M2 as the differential pair. M1 gate connects to Vinp, M2 gate connects to Vinn, and their sources join at the tail node. Use PMOS M3/M4 as the active load from VDD to the two drain/output nodes. M3 is diode-connected on the Voutp side to form the PMOS mirror reference, with M3 drain/gate tied to Voutp. M4 gate ties to M3 gate and M4 drain connects to Vout. M3 and M4 sources connect to VDD. Use NMOS M5 as the tail current source from the shared source tail node to ground, with M5 gate driven by Vbias, drain at the tail node, source/bulk at ground.

Required devices:
M1: NMOS differential input transistor, drain at Voutp, gate Vinp, source tail node, bulk tied to source.
M2: NMOS differential input transistor, drain at Vout, gate Vinn, source tail node, bulk tied to source.
M3: PMOS diode-connected active load, source/bulk VDD, drain/gate Voutp.
M4: PMOS mirror load, source/bulk VDD, gate Voutp, drain Vout.
M5: NMOS tail current source, drain tail node, gate Vbias, source/bulk ground.

Biasing assumptions: VDD is 5 V. Drive Vinp and Vinn explicitly with independent DC voltage sources at 2.5 V for balanced operation. Drive Vbias explicitly with an independent DC source high enough to turn on M5, for example about 1.2 V to 1.5 V, giving VGS above the 0.7 V NMOS threshold while preserving headroom for M1/M2. The common source tail node should settle below the 2.5 V inputs so M1/M2 conduct, and Voutp/Vout should settle below VDD enough for the PMOS loads to conduct.

MOS models: Use the requested NMOS model with Vth=0.7 V, kp=100u, W=50u, L=1u, and PMOS model with Vth=-0.7 V, kp=50u, W=100u, L=1u. Connect MOS bulks to their sources.

Analyses: Include `.op` to verify the DC operating point. The operating point should show nonzero drain current in M1, M2, M3, M4, and M5. With Vinp=Vinn=2.5 V, the two branches should be approximately balanced, and Voutp and Vout should both sit at valid high-impedance gain/load nodes rather than rails.

Output constraints: The final netlist must contain the exact input nodes Vinp, Vinn, and Vbias, each driven by an explicit independent voltage source. The final netlist must contain the exact output nodes Voutp and Vout. Use no subcircuits or hidden floating inputs.
