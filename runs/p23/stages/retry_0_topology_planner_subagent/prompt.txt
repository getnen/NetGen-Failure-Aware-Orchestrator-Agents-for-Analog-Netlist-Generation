Objective: Implement a 5 V single-supply Wien Bridge oscillator whose output node Vout develops a visible sustained sinusoidal oscillation during `.tran 1u 5m`. There are no external input nodes. The final netlist must include node Vout.

Topology: Use an op-amp-like high-gain voltage-controlled gain stage powered from VDD=5 V and ground, configured as a non-inverting Wien Bridge oscillator around a mid-supply reference. Create a mid-supply node Vref at approximately 2.5 V with a resistor divider and bypass capacitor. The Wien positive-feedback network connects from Vout to the op-amp non-inverting input Vinp using a series RC branch, with a parallel RC branch from Vinp to Vref. This gives zero phase shift and 1/3 feedback gain at the oscillation frequency. The negative-feedback network connects from Vout to the op-amp inverting input Vinn and from Vinn to Vref, setting closed-loop gain slightly above 3 for startup.

Required devices:
- Independent DC supply source VDD from VDD to 0, value 5 V.
- Mid-supply bias divider: two equal resistors from VDD to Vref and Vref to 0, plus a bypass capacitor from Vref to 0.
- High-gain op-amp/gain-stage element powered or behaviorally limited to the 0 to 5 V rails. Its internal differential input nodes must be named Vinp and Vinn so startup asymmetry can be initialized or injected there.
- Wien network:
  - Rw1 and Cw1 in series from Vout to Vinp.
  - Rw2 and Cw2 in parallel from Vinp to Vref.
  - Choose Rw1=Rw2 and Cw1=Cw2 for a clean oscillation frequency, e.g. R=10 kOhm and C=10 nF, giving about 1.59 kHz.
- Negative feedback:
  - Rg from Vinn to Vref.
  - Rf from Vout to Vinn.
  - Set non-inverting gain slightly above 3, e.g. Rg=10 kOhm and Rf=22 kOhm to start oscillation. If amplitude stabilization is included, use diodes or a nonlinear limiter in the feedback/gain stage so Vout remains bounded inside the 0 to 5 V supply.
