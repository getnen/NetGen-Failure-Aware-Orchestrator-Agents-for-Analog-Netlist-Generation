**Topology Summary**

Use a single-supply transistor-level Wien Bridge oscillator powered by `VDD vdd 0 DC 5`, with output node exactly `Vout`. The oscillator core should be a real analog non-inverting gain stage biased around `Vmid`, not a behavioral op-amp, VCVS, or CMOS rail-switching core.

Recommended topology:

- `Vmid` generated by a 5 V resistor divider and bypass capacitor.
- Wien positive feedback from `Vout` to explicit non-inverting input node `Vinp`.
- Resistive negative feedback from `Vout` to explicit inverting input node `Vinn`.
- Transistor differential input pair sensing `Vinp - Vinn`.
- Opposite-polarity active load for voltage gain.
- Modest analog output stage driving `Vout`, biased for linear swing around `Vmid`.

**Device / Function List**

- `VDD`: 5 V supply source, exact form `VDD vdd 0 DC 5`.
- `Rmid_top`, `Rmid_bot`: equal resistors from `vdd` to `Vmid` to `0`.
- `Cmid`: decoupling capacitor from `Vmid` to `0`.
- Differential pair:
  - Prefer NMOS input pair with sources tied to an NMOS tail current source.
  - Gates are `Vinp` and `Vinn`.
  - PMOS current mirror or active load provides high gain.
  - Bulks tied to sources.
- Bias current devices:
  - Use resistor or diode-connected MOS biasing from `vdd`/`0`; no ideal current-source macros unless implemented with primitive MOS/resistor devices.
- Output stage:
  - A simple common-source or source-follower/buffered analog stage biased near mid-supply.
  - Add output load resistance and small compensation/load capacitance if needed for transient conditioning.
- Wien bridge:
  - Series branch from `Vout` to `Vinp`: `Cw1` then `Rw1`, or `Rw1` then `Cw1`.
  - Shunt branch from `Vinp` to `Vmid`: `Rw2` in parallel with `Cw2`.
  - Use `Rw1 = Rw2 = 10k`, `Cw1 = Cw2 = 10n`.
- Negative feedback:
  - `Rg = 10k` from `Vinn` to `Vmid`.
  - Startup feedback resistance from `Vout` to `Vinn` around `22k` to `24k`, giving closed-loop gain about `3.2` to `3.4`.
- Gentle amplitude stabilization:
  - Use primitive antiparallel diodes or diode-connected MOS/resistor elements in the feedback path only.
  - They should reduce effective gain as amplitude grows, not hard-clamp `Vout`.

**Bias / Reference Strategy**

Center the whole oscillator around `Vmid ≈ 2.5 V`.

- `Vinp` must be DC-referenced to `Vmid` through the Wien shunt resistor.
- `Vinn` must be DC-referenced to `Vmid` through `Rg`.
- The differential pair input common-mode should operate around 2.5 V.
- The output stage should idle near 2.5 V and allow roughly 1 Vpp to 3 Vpp swing without hitting rails.
- Use `.ic V(Vinp)=2.51 V(Vinn)=2.49 V(Vout)=2.5` or similar explicit startup asymmetry.

**Signal / Feedback Path**

Positive feedback path:

`Vout -> series Wien RC -> Vinp -> parallel Wien RC to Vmid`

At approximately:

`f0 = 1 / (2*pi*10k*10n) ≈ 1.59 kHz`

the Wien network has near-zero phase shift and about `1/3` attenuation.

Negative feedback path:

`Vout -> Rf / nonlinear stabilization network -> Vinn -> Rg -> Vmid`

The transistor amplifier should be wired non-inverting overall: a rise at `Vinp` relative to `Vinn` must cause `Vout` to rise. If the differential pair plus gain/output stages invert once too many or too few times, the Wien loop will not oscillate.

**SPICE / Netlist Constraints**

- Final file should be `p23.cir`.
- Must include node `Vout`.
- Must include explicit nodes `Vinp`, `Vinn`, and `Vmid`.
- Must include exactly:
  - `.tran 1u 5m`
- Use primitive MOSFETs, resistors, capacitors, diodes, and voltage sources only.
- Do not use `E`, `F`, `G`, `H`, behavioral sources, ideal op-amp macros, or hidden subcircuits.
- Define all MOS/diode models explicitly in the same file.
- Tie MOS bulks to sources.
- Do not leave `Vinp`, `Vinn`, or internal gain nodes floating.
- Do not rely on commented-out devices for biasing, startup, or feedback.

**Failure Risks To Avoid**

- Insufficient loop gain: the Wien network attenuates by about `1/3`, so startup closed-loop gain must be slightly above `3`.
- Wrong feedback polarity: verify that `Vinp` is the non-inverting input and `Vinn` is the inverting input in the realized transistor gain chain.
- Ground-centered bridge: the Wien bridge must return to `Vmid`, not ground.
- Floating input nodes: both `Vinp` and `Vinn` need DC paths to `Vmid`.
- Over-aggressive limiting: hard clamps or rail-switching output stages can kill sinusoidal startup or create non-periodic clipping.
- Too weak transistor gain stage: a marginal differential pair/load can reproduce the prior tiny ripple failure.
- Output stuck near rail: bias the output stage so `Vout` naturally rests near 2.5 V.
- Excessive capacitance on high-impedance gain nodes: can shift phase enough to suppress oscillation or move the dominant frequency far from 1.6 kHz.
