Objective: Build a startup-capable single-supply Wien Bridge oscillator using VDD = 5 V and output node Vout. The transient analysis must show a visible sustained periodic oscillation at Vout during `.tran 1u 5m`, especially in the latter half of the waveform.

Topology: Use a non-inverting op-amp-style gain stage biased around mid-supply, with a Wien Bridge positive feedback network from Vout to the non-inverting input. Use a separate resistive negative feedback path setting closed-loop gain slightly above 3 at startup, then limit amplitude gently with antiparallel diodes or MOS-connected nonlinear elements in the feedback leg. This is a topology-level redesign from weak-loop attempts: use a clean single-ended voltage amplifier macro built from dependent sources or a transistor/op-amp gain stage, not a marginal complementary-input core.

Required nodes:
Vout must be the oscillator output node.
No external input nodes are required.

Required supplies and references:
Use `VDD vdd 0 DC 5`.
Create a mid-supply reference node, for example `Vmid`, using two equal resistors from vdd to ground and decouple it with a capacitor to ground. The op-amp input common mode and feedback network must be centered around Vmid, not ground, because this is a 5 V single-supply oscillator.

Core signal path:
The amplifier output is Vout.
The non-inverting input node, e.g. `Vinp`, receives frequency-selective positive feedback through the Wien network:
- Series branch from Vout to Vinp: capacitor then resistor, or resistor then capacitor.
- Shunt branch from Vinp to Vmid: resistor and capacitor in parallel.
Use equal R and equal C values for the Wien bridge so its phase shift is near zero and attenuation is about 1/3 at oscillation frequency.

Recommended Wien values:
Use R = 10 kOhm and C = 10 nF for both Wien arms.
Expected oscillation frequency is approximately:
f0 = 1 / (2*pi*R*C) ≈ 1.59 kHz.
This gives several cycles within 5 ms and should be easy for the evaluator to detect.

Amplifier and negative feedback:
Use an op-amp-like high-gain voltage-controlled voltage source or a simple transistor gain stage with enough gain. The easiest robust netlist-first implementation is:
- High-gain differential VCVS sensing `Vinp` and `Vinn`.
- Output limited or biased so Vout swings around Vmid within the 0 to 5 V rails.
- If using ideal controlled sources, include output resistance/load capacitance so transient behavior is well-conditioned.

Set the negative feedback divider from Vout to Vinn to Vmid for a non-inverting closed-loop gain slightly greater than 3 at startup:
Gain = 1 + Rf/Rg.
Choose Rg = 10 kOhm from Vinn to Vmid.
Choose Rf effective startup value around 22 kOhm to 24 kOhm from Vout to Vinn, giving gain about 3.2 to 3.4.
Add nonlinear stabilization so effective gain drops toward 3 as amplitude grows. A practical implementation:
- Put a main feedback resistor, e.g. 22 kOhm to 24 kOhm, from Vout to Vinn.
- Add antiparallel diodes or diode-connected MOSFET/resistor branch in parallel with part of the feedback path so large signals reduce gain gently.
Avoid hard clipping as the only amplitude control; it can prevent clean periodicity.

Startup requirements:
Initialize internal op-amp input nodes with a slight mismatch so the evaluator can start oscillation. For example:
`.ic V(Vinp)=2.51 V(Vinn)=2.49 V(Vout)=2.5`
Do not leave Vinp or Vinn floating. Vinp must be connected by the Wien bridge to Vout and Vmid. Vinn must be connected by the negative feedback divider to Vout and Vmid.

Biasing assumptions:
Vout should idle near 2.5 V and oscillate around 2.5 V.
The gain stage must have enough small-signal loop gain to overcome the Wien network attenuation at startup.
If MOSFETs are used, connect each bulk to its source and respect NMOS Vth = 0.7 V, PMOS Vth = -0.7 V. Prefer nominal sizing unless needed for gain.
If a behavioral or dependent-source op-amp is used, keep it explicitly powered or rail-limited to 0 V and 5 V if supported by the simulator.

Analysis:
Include exactly the required transient analysis:
`.tran 1u 5m`

Expected verification behavior:
Vout should develop a sinusoidal or lightly compressed periodic waveform centered near 2.5 V.
The latter half of the transient should have clearly detectable peaks, non-trivial amplitude, and a dominant frequency near 1.6 kHz.
A target usable Vout amplitude is at least several hundred millivolts peak-to-peak; 1 Vpp to 3 Vpp is acceptable as long as it remains periodic and within the 0 to 5 V supply range.

Output constraints:
The final netlist must contain node `Vout`.
Use no hidden subcircuits or undefined models.
Do not rely on commented-out devices or floating startup nodes.
The oscillator must be self-contained with explicit VDD, Vmid generation, gain stage, Wien feedback network, amplitude stabilization, `.ic` startup asymmetry, and `.tran 1u 5m`.
